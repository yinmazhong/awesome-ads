# 海外主要玩家 (Overseas Players)

## Google

### 广告产品矩阵

| 产品 | 类型 | 说明 |
|------|------|------|
| **Google Search Ads** | 搜索广告 | 核心收入来源，关键词竞价 |
| **YouTube Ads** | 视频广告 | 前贴片、信息流、Shorts |
| **Google Display Network** | 展示广告 | 覆盖 200 万+ 网站 |
| **Google Shopping** | 购物广告 | 商品 Feed 驱动 |
| **Google AdMob** | 移动广告 | App 内广告变现 |
| **Performance Max (PMax)** | 全自动广告 | AI 驱动，跨渠道投放 |
| **DV360** | DSP | 程序化购买平台 |
| **Google Ad Manager** | Ad Server + SSP | 媒体广告管理 |

### 核心技术优势

- **搜索意图数据**: 全球最大的搜索引擎，用户意图数据无可替代
- **AI/ML 能力**: 深度学习在广告中的先驱 (Wide&Deep, BERT)
- **全链路闭环**: 从 DSP 到 Ad Exchange 到 SSP 的完整生态
- **Performance Max**: AI 全自动投放的标杆产品

### 广告收入
- 2023 年广告收入: ~$2,380 亿 (占 Alphabet 总收入 ~77%)

## Meta (Facebook)

### 广告产品矩阵

| 产品 | 平台 | 说明 |
|------|------|------|
| **Facebook Ads** | Facebook | 信息流、Stories、Reels |
| **Instagram Ads** | Instagram | 图片/视频/Stories/Reels |
| **Messenger Ads** | Messenger | 消息广告 |
| **Audience Network** | 第三方 App | 联盟广告 |
| **Advantage+** | 全平台 | AI 自动化投放 |
| **WhatsApp Business** | WhatsApp | 商业消息 (增长中) |

### 核心技术优势

- **社交图谱**: 30 亿+ 用户的社交关系数据
- **兴趣定向**: 基于社交行为的精准兴趣定向
- **Lookalike Audiences**: 业界标杆的相似人群扩展
- **Advantage+ Shopping**: AI 驱动的电商广告自动化

### iOS ATT 影响与应对

```
影响:
  - 2021 年 iOS ATT 导致 Meta 广告收入损失 ~$100 亿/年
  - 定向精准度下降，归因困难

应对:
  - 大力投资 AI/ML (Advantage+ 系列)
  - 推动 Conversions API (S2S 归因)
  - 建设聚合事件衡量 (AEM)
  - 投资端侧 AI 能力
```

### 广告收入
- 2023 年广告收入: ~$1,310 亿

## 行业趋势

### 零售媒体网络 (Retail Media Network)

```
零售商利用自有电商数据和流量变现:
  - Amazon Ads (先驱)
  - Walmart Connect
  - Target Roundel
  - Instacart Ads
  - Kroger Precision Marketing

优势: 购买意图数据 + 闭环归因
增速: 2024 年全球零售媒体广告 ~$1,400 亿
```

### CTV/OTT 广告

```
联网电视广告快速增长:
  - Netflix (2022 年推出广告套餐)
  - Disney+ (广告套餐)
  - YouTube TV
  - Roku
  - Amazon Fire TV

特点: 大屏体验 + 数字化定向 + 可衡量
```
