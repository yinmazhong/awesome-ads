# 在线广告发展史

## 一句话概述

在线广告从 1994 年第一个 Banner 广告诞生至今，经历了展示广告、搜索广告、程序化购买、移动广告、信息流广告等多个阶段，正在进入 AI 驱动 + 隐私优先的新时代。

---

```

 │           │           │           │           │           │           │           │

```

## 第一阶段：展示广告萌芽期 (1994–1999)

### 关键事件
- **1994**: AT&T 在 HotWired 网站投放第一个 Banner 广告，CTR 高达 44%
- **1995**: DoubleClick 成立，开创广告服务器 (Ad Server) 模式
- **1996**: 雅虎开始销售展示广告，CPM 模式确立
- **1998**: GoTo.com (后更名 Overture) 推出付费搜索概念

### 特征
- 广告形式单一：以 Banner (横幅) 为主
- 购买方式：人工谈判，按 CPM 计费
- 定向能力弱：基本只能按频道/页面投放
- 效果衡量粗糙：以曝光量为主

---

## 第二阶段：搜索广告崛起 (2000–2006)

### 关键事件
- **2000**: Google 推出 AdWords (后更名 Google Ads)
- **2002**: Google 引入 CPC 竞价模式 + 质量分 (Quality Score)
- **2003**: Overture 被雅虎收购
- **2006**: Google 以 31 亿美元收购 DoubleClick

### 核心创新
- **搜索广告模型**: 用户主动搜索 → 展示相关广告 → 高意图流量
- **CPC 竞价**: 从 CPM 转向按点击付费，广告主风险降低
- **质量分机制**: 不仅看出价，还看广告质量，平衡用户体验与商业化
- **长尾变现**: 搜索广告让中小广告主也能参与

### 中国市场
- **2001**: 百度成立，推出竞价排名
- **2005**: 百度上市，搜索广告成为中国互联网主要商业模式

---

## 第三阶段：广告网络化 (2003–2009)

### 关键事件
- **2003**: Google 推出 AdSense，开创内容联盟模式
- **2005**: 广告网络 (Ad Network) 大量涌现
- **2007**: Facebook 推出 Social Ads，开启社交广告
- **2008**: 苹果推出 App Store，移动生态萌芽

### 核心概念
- **Ad Network (广告网络)**: 聚合中小媒体流量，统一售卖给广告主
  - 代表：Google AdSense, 百度联盟, ValueClick
- **Vertical Ad Network**: 按行业/内容垂直聚合
- **问题**: 不透明、层层转卖、流量质量参差不齐

### 定向技术进步
- 行为定向 (Behavioral Targeting) 出现
- Cookie 追踪技术成熟
- 受众细分开始替代内容定向

---

## 第四阶段：程序化购买时代 (2009–2015)

### 关键事件
- **2009**: RTB (Real-Time Bidding) 概念提出，第一个 Ad Exchange 上线
- **2010**: DSP 平台涌现 (The Trade Desk, MediaMath, Turn)
- **2011**: Google 推出 DoubleClick Bid Manager (DBM)
- **2012**: Facebook Exchange (FBX) 支持 RTB
- **2013**: 程序化直接购买 (Programmatic Direct) 出现
- **2014**: Header Bidding 技术出现，挑战 Google 的 Waterfall 模式

### 核心架构

```
广告主 → DSP → Ad Exchange ← SSP ← 媒体
              ↕
            DMP (数据)
```

- **DSP (Demand-Side Platform)**: 广告主侧的程序化购买平台
- **SSP (Supply-Side Platform)**: 媒体侧的程序化售卖平台
- **Ad Exchange**: 连接 DSP 和 SSP 的交易市场
- **DMP (Data Management Platform)**: 数据管理平台，提供人群数据

### RTB 流程
1. 用户访问页面 → 媒体向 SSP 发送广告请求
2. SSP 向多个 DSP 发送竞价请求 (Bid Request)
3. DSP 在 ~100ms 内决策并出价 (Bid Response)
4. Ad Exchange 选出最高出价者
5. 胜出 DSP 的广告素材展示给用户

### 中国市场
- **2012**: 品友互动、悠易互通等国内 DSP 涌现
- **2013**: 百度 BES、腾讯广点通、阿里 TANX 上线
- **2014**: 国内程序化购买市场快速增长

---

## 第五阶段：移动广告爆发 (2012–2016)

### 关键事件
- **2012**: Facebook 移动广告收入从 0 到占比 30%
- **2013**: 移动广告支出首次超过桌面
- **2014**: 原生广告 (Native Ads) 成为主流形式
- **2015**: 应用安装广告 (App Install Ads) 爆发
- **2016**: 视频广告快速增长

### 移动广告特有挑战
- **设备标识**: IDFA (iOS) / GAID (Android) / OAID (国内安卓)
- **归因**: 安装归因 (Install Attribution) 成为核心需求
  - MMP 平台: AppsFlyer, Adjust, Branch, 热云
- **深度链接**: DeepLink / Deferred DeepLink
- **SDK 集成**: 广告 SDK 嵌入 App

### 广告形式演进
- Banner → 插屏 (Interstitial) → 原生 (Native) → 激励视频 (Rewarded Video) → 开屏 (Splash)

### 中国市场
- **2013**: 微信朋友圈广告上线
- **2015**: 今日头条信息流广告快速增长
- **2016**: 短视频广告 (抖音/快手) 开始崛起

---

## 第六阶段：信息流与短视频广告 (2016–2020)

### 关键事件
- **2016**: 抖音上线，短视频广告开始爆发
- **2017**: 信息流广告超越搜索广告成为最大广告形式 (中国市场)
- **2018**: TikTok 全球化，短视频广告模式输出海外
- **2019**: 直播电商广告兴起
- **2020**: 疫情加速线上广告增长

### 核心变化
- **推荐驱动**: 从"人找信息"到"信息找人"，广告融入推荐流
- **智能出价**: oCPM/oCPC 成为主流，平台接管优化
- **短视频创意**: 15s/60s 视频成为核心广告形式
- **效果广告主导**: 品牌广告占比持续下降

### 中国市场格局
- **字节跳动**: 巨量引擎成为最大广告平台之一
- **腾讯**: 微信生态广告 (朋友圈 + 公众号 + 小程序)
- **阿里**: 电商广告持续增长 (直通车/钻展/超级推荐)
- **快手**: 磁力引擎发力下沉市场

---

## 第七阶段：隐私保护时代 (2018–至今)

### 关键事件
- **2018**: GDPR 生效 (欧盟)
- **2020**: CCPA 生效 (加州); Apple 宣布 ATT 框架
- **2021**: iOS 14.5 ATT 正式实施，IDFA 获取率暴跌至 ~20%
- **2021**: 中国《个人信息保护法》生效
- **2022**: Google 宣布 Chrome 将淘汰第三方 Cookie (后推迟)
- **2023**: Google Privacy Sandbox / Topics API 推进
- **2024**: Chrome 第三方 Cookie 逐步淘汰

### 影响
- **归因困难**: iOS 端归因准确性大幅下降
- **定向受限**: 第三方数据可用性降低
- **第一方数据价值飙升**: 广告主自有数据成为核心资产
- **隐私计算兴起**: 联邦学习、差分隐私、TEE 等技术应用

### 行业应对
- SKAdNetwork / SKAN (Apple 的隐私归因方案)
- Privacy Sandbox / Topics API (Google 的 Cookie 替代方案)
- 服务端归因 (Server-to-Server)
- 数据清洁室 (Data Clean Room)
- 隐私增强技术 (PETs)

---

## 第八阶段：AI 驱动广告 (2023–未来)

### 关键趋势
- **AIGC 创意**: AI 生成广告素材 (文案/图片/视频)
  - Google Performance Max, Meta Advantage+
  - 国内: 巨量创意、腾讯混元
- **大模型应用**: 
  - 智能投放助手 (自然语言创建广告计划)
  - 意图理解增强 (搜索广告)
  - 智能客服与转化
- **全自动化投放**: 从"人工优化"到"AI 托管"
  - Google PMax, Meta ASC (Advantage+ Shopping Campaigns)
- **端侧 AI**: 设备端模型实现隐私保护下的个性化
